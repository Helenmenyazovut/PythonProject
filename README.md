# PythonProject


Идея проекта:
    Тюнер для гитары с визуализацией гитары и струн, а также "начальным обучением": аккорды и бой.
Делаю одна: Щербакова Елена Николаевна, группа Б05-524.
(https://github.com/Helenmenyazovut/PythonProject)
Библиотеки:
    Из основных: Numpy, MatplotLib.
    Из ещё незнакомых: PyAudio, seeFretboard, Tkinter/kivi(ещё посмотрю, что будет оптимальнее)
Фичи:
    Определяет по звуку, правильно ли настроена струна и показывает отклонение
    Показывает примеры боя и аккордов (во вкладке Обучение)
Обновление:
    Добавила файлы. Для того, чтоб запустить проект, создайте виртуальное окружение venv, установите зависимости из requirements.txt и системные библиотеки для pyaudio.
    Запустите python main.py. Откроется главное окно с тюнером. При звучании струны будет показывать отклонение на линеечке, а на отображении гитары показывать ту самую струну нужным цветом в зависимости от настройки (в целом работает, но я ещё буду дорабатывать). Во вкладке обучение есть аккорды и бой, можно в них зайти и увидеть разные аккорды на гитарном грифе и виды боя.
